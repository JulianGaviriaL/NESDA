2. EFFECTIVE ECHO SPACING CALCULATION (3 Stages)

Overview
EffectiveEchoSpacing calculated from Philips PAR files using the water-fat shift method, following standard practice in 
the neuroimaging community for Philips 3T data.
https://neurostars.org/t/consolidating-epi-echo-spacing-and-readout-time-for-philips-scanner/4406
https://github.com/rordenlab/dcm2niix/issues/377

Implementation
Stage 1: Extract Water-Fat Shift
Python
# Search PAR file for "Water Fat shift [pixels]"
water_fat_shift = 8.5  # example value in pixels

Stage 2: Extract Reconstruction Matrix
Python
# Search PAR file for "Recon resolution (x, y)"
recon_matrix_pe = 80  # y-dimension (phase-encoding direction)
Stage 3: Calculate Bandwidth per Pixel
Python
bandwidth_factor = 434.215  # Hz (water-fat chemical shift at 3T)
bandwidth_per_pixel_hz = 434.215 / water_fat_shift
Constant 434.215 Hz:

Water-fat chemical shift at 3 Tesla for Philips systems
Derived from: γ (42.57 MHz/T) × B₀ (3T) × δ (3.5 ppm) ≈ 447 Hz theoretical
Philips uses 434.215 Hz (empirically calibrated)
Stage 4: Calculate EffectiveEchoSpacing
Python
ees = 1.0 / (bandwidth_per_pixel_hz * recon_matrix_pe)
EffectiveEchoSpacing = round(ees, 8)  # seconds
